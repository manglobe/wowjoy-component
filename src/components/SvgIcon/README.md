# SvgIcon

SvgIcon 组件

## 使用

- 基础用法

```jsx
import React from "react";
import { SvgIcon } from "wowjoy-component";

const Foo = () => <SvgIcon></SvgIcon>;
```

## APIs

| 属性          | 描述           |  类型  | 默认值 |
| ------------- | :------------- | :----: | :----: |
| className     | 顶层样式 class | string |        |
| defaultStyles | 顶层默认样式   | string |        |
| children      | 内容           |  node  |        |
